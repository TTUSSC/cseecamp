Arduino

Arduino 是一個結合軟硬體並且開源的開發平台。 讓我們可以使用簡易的方式，輕鬆地透過程式去控制硬體。 如經已應用於機器人、物聯網、自動化控制等等領域。

Arduino 官網：
https://www.arduino.cc/

這次營隊所使用的是 Arduino UNO 的開發板，是屬於 Arduino 系列的開發板中最基礎也是最受初學者歡迎的一款。 Arduino Uno 具有 14 個數位 I/O 引腳，6 個類比輸入引腳，可以連接到各種擴展板和其他電路，並且可以通過 B 型 USB 線(和列表機相同的方口 USB 線)與 Arduino IDE 進行程式編寫。
